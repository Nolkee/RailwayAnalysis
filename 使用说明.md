# 川渝地区轨道交通客流数据分析展示系统 - 使用说明

## 项目简介

本项目是一个基于C++和Qt的轨道交通客流数据分析系统，具备以下功能：
- 数据导入和管理
- 统计分析
- 时间序列预测
- 聚类分析
- 可视化展示

## 环境要求

- Windows 10/11
- Qt 6.9.1 (已安装)
- MinGW 8.1.0 (已安装)

## 编译方法

### 方法一：使用Qt Creator (推荐)

1. **打开Qt Creator**
   - 启动Qt Creator IDE
   - 选择 "文件" → "打开文件或项目"
   - 选择项目根目录下的 `RailwayAnalysis.pro` 文件

2. **配置项目**
   - Qt Creator会自动检测Qt版本
   - 选择MinGW编译器
   - 点击 "配置项目"

3. **编译运行**
   - 点击左下角的绿色三角形按钮编译并运行
   - 或按 `Ctrl+R` 快捷键

### 方法二：命令行编译 (如果遇到兼容性问题)

如果Qt Creator编译成功，但命令行编译失败，这是正常的。Qt Creator会自动处理版本兼容性问题。

## 数据文件

确保以下CSV数据文件在项目根目录：
- `客运站点（站点名称、站点编号、备注）.csv`
- `列车表（列车编码、列车代码、列车运量）.csv`
- `运营线路客运站（运营线路编码、站点id、线路站点id、上一站id、运营线路站间距离 、下一站id、运输距离、线路代码）.csv`
- `高铁客运量（成都--重庆）（运营线路编码、列车编码、站点id、日期、到达时间、出发时间、上客量、下客量等，起点站、终点站、票价、收入等）(3).csv`

## 功能说明

### 1. 数据管理
- 自动加载CSV数据文件
- 数据验证和清洗
- 数据导出功能

### 2. 统计分析
- 站点客流量统计
- 列车运营统计
- 时间分布分析
- 收入分析

### 3. 聚类分析
- **站点聚类**：根据客流量、票价等特征将站点分类
- **列车聚类**：识别不同类型的列车运营模式
- **时间聚类**：发现客流的时间分布规律

### 4. 预测分析
- 时间序列预测
- 客流趋势分析
- 模型评估

### 5. 可视化
- 多种图表类型（柱状图、折线图、散点图、饼图）
- 交互式数据表格
- 聚类结果可视化

## 聚类模型应用

### 站点聚类的作用：
- **高客流量站点**：需要增加列车频次，优化换乘设施
- **中等客流量站点**：维持现有运营水平
- **低客流量站点**：优化运营时间，降低成本

### 列车聚类的作用：
- 识别高速列车、动车组、城际列车等不同类型
- 优化运力配置和调度策略
- 制定差异化定价策略

### 时间聚类的作用：
- 识别高峰时段、低谷时段、稳定时段
- 为运力调度提供数据支持
- 优化运营时间安排

## 故障排除

### 常见问题：

1. **Qt路径错误**
   - 确保Qt安装在 `E:\Download\QT\6.9.1\mingw_64`
   - 在Qt Creator中检查编译器设置

2. **编译错误**
   - 使用Qt Creator而不是命令行编译
   - 确保所有源文件都存在

3. **数据文件缺失**
   - 确保CSV文件在项目根目录
   - 检查文件名是否正确

4. **运行时错误**
   - 确保Qt运行时库在PATH中
   - 检查数据文件格式

## 技术支持

如果遇到问题，请检查：
1. Qt版本是否为6.9.1
2. MinGW版本是否为8.1.0
3. 所有源文件和头文件是否存在
4. CSV数据文件是否完整
